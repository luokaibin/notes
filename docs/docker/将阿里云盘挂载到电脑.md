### docker-compose 配置

```yml
version: '2.1'
services:
  webdav_aliyundriver: # 服务的名称
    restart: always # 代表只要docker启动，那么这个容器就跟着一起启动
    image: zx5253/webdav-aliyundriver:2021-08-22 # 指定镜像路径
    container_name: webdav_aliyundriver # 指定容器名称
    ports:
      - 7780:8080 # 指定端口号的映射
    environment: # 指定环境信息
      ALIYUNDRIVE_REFRESH_TOKEN: 4d22e497622944e7a1249ec9b1d95748 # 指定MySQL的ROOT用户登录密码
      ALIYUNDRIVE_AUTH_PASSWORD: b4qfbgkd1z9LHC # 镜像启动时创建的数据库的名称
      JAVA_OPTS: -Xmx1g # 创建新用户和设置该用户的密码一起使用。将为该用户授予该MYSQL_DATABASE变量指定的数据库的超级用户权限。这两个变量都是创建用户所必需的
      TZ: Asia/Shanghai # 指定时区
    volumes: # 配置数据卷
      - /etc/localtime:/etc/localtime # 映射数据卷 宿主机路径:容器路径
      - /data/aliyun/driver/:/etc/aliyun-driver/
```

### 启动

```
docker-compose up -d
```
