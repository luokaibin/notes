> 一款简洁、轻量的单栏 Hexo 主题。

> 开发不易，欢迎 [点击此处](https://yfun.top/?ref=donate&from=github) 支持作者。

## 功能特性

- [x] 代码高亮
- [x] 内置多语言
- [x] 多款评论插件
- [x] 自定义静态资源 CDN
- [x] 站内搜索
- [x] 页脚备案信息
- [x] 网页统计
- [x] 支持脚注语法
- [x] 深色模式
- [x] 响应式设计
- [x] 文章目录
- [x] 阅读体验优化
- [x] Mermaid 流程图
- [x] [HexoPlusPlus](https://hexoplusplus.js.org) 适配
- [x] 友链页面（JSON / YAML）
- [x] 方便排版的 Tag 插件
- [x] Twikoo / Waline 评论懒加载
